# ART FLUX DESIGNER Website

This is a premium portfolio website for Art Flux Designer, built with HTML, CSS, JavaScript, and Tailwind CSS.

## How to Run

Since this project does not use Node.js, you can simply open the files directly in your web browser.

1.  Navigate to the folder `new ishan`.
2.  Double-click on `index.html` to open the website.

## Pages Included

1.  **Home (`index.html`)**: Main landing page with a hero section and customer counters.
2.  **About (`about.html`)**: Information about the team and mission.
3.  **Services (`services.html`)**: List of services provided.
4.  **Projects (`projects.html`)**: Gallery of projects and designs.
5.  **Contact (`contact.html`)**: Contact form and business details.

## Technologies

-   HTML5
-   Tailwind CSS (via CDN)
-   JavaScript (Vanilla)
-   Google Fonts (Inter)
